## About page
Project developed with the library [React](https://es.reactjs.org/) using [tailwindcss](https://tailwindcss.com/) for the styles

## Installation
Clone the project

```bash
git clone https://github.com/ESPENSER159/page404_React.git && cd page404_React
```
Install dependences
```bash
npm i
```
Run the project
```bash
npm run start
```
